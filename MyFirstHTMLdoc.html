Attributes

Attributes in the HTML language provide additional information about an element
or a tag, while also modifying them. The value, if provided, modifies the atribute.
For example, the value "center" indicates that the paragraph within the p elements
should be positioned in the center. In another exampe, measurements in pixels and
percentages can also be used inside the hr tag using the px and % indicators to
change the length of a horizontal line. Moreover, mutiple attributes can be typed
within a tag.

Images

The img tag is used to place an image into an html document. It only has
attributes and is devoid of a closing tag. The image's URL can be defined using
the src attribute. To define the size of the image, use the height and width
attributes, values can be specified in pixels or as percentages. In addition,
the border attribute can be used to add a border depending on the px value.

Links

You can add links through the a tag. The href attribute will define the link's
destination address. To link an image to another document, nest the img tag inside
the a tags. To have the linked document open in a new tab/window, use the "_blank"
value after the target attribute. The link to SoloLearn is below.

Lists

Ordered lists start with the ol tag, and each item is defined by the li tag.
Below is an example of an ordered list.

  1. Apples
  2. Oranges
  3. Bananas
  4. Grapes

Unordered lists start with the ul tag. Instead of numbers, each item will
be marked by a bullet.
Below is an example of an unordered list.

  • Dogs
  • Cats
  • Monkeys
  • Fishes

Tables

In HTML, tables are defined by the table tag. Tables are divided into rows with
the tr tag. Moreover, columns can be created out of rows by the td tag. Note that
columns are also called table data, which can contain all kinds of HTML contents,
including text, images, lists, and even other tables.
